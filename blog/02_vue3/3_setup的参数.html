  • 保证prop是单向数据流

每次父级组件发生变更时，子组件中所有的 prop 都将会刷新为最新的值。这意味着不应该在一个子组件内部改变 prop。
防止从子组件意外变更父级组件的状态会采取以下两种方式 非语法糖

  // 第一种
  props:['name','age'], 
  setup(props) {
    const myName = ref(props.title)
    const myAge = ref(props.age)
  } 
  // 第二种 如果改值需要一些处理 可以利用计算属性 
  props: ['name','age'],
  computed: {
    myName() {
      return this.name
    }
  }
