viper

配置管理工具，提供了一种简单而灵活的方式来读取、解析和使用配置文件。

简单使用案例

  1. 安装依赖
go get github.com/spf13/viper

  port: 3306
  username: root
  password: password123

  1. 使用viper读取文件以及使用其中的值
package main

import (
	"fmt"

	"github.com/spf13/viper"
)

func main() {
	// 设置配置文件名和路径
	viper.SetConfigName("config")
	viper.SetConfigType("yaml")
	viper.AddConfigPath(".")
	
	// 读取配置文件
	err := viper.ReadInConfig()
	if err != nil {
		fmt.Printf("无法读取配置文件: %s\n", err)
		return
	}
	
	// 获取配置项的值
	appName := viper.GetString("app.name")
	appVersion := viper.GetString("app.version")
	dbHost := viper.GetString("database.host")
	dbPort := viper.GetInt("database.port")
	dbUsername := viper.GetString("database.username")
	dbPassword := viper.GetString("database.password")
	
	// 使用配置项的值
	fmt.Printf("应用名称: %s\n", appName)
	fmt.Printf("应用版本: %s\n", appVersion)
	fmt.Printf("数据库主机: %s\n", dbHost)
	fmt.Printf("数据库端口: %d\n", dbPort)
	fmt.Printf("数据库用户名: %s\n", dbUsername)
	fmt.Printf("数据库密码: %s\n", dbPassword)
}

配置项

把配置值读入 Viper

  • Viper 支持多种方式读入配置：
    • 设置默认配置值
    • 从配置文件读取配置
    • 从环境变量读取配置
    • 监控并重新读取配置文件
    • 从 io.Reader 读取配置
    • 从命令行参数读取配置
    • 从远程 key/value 存储读取配置

设置默认配置值

Viper 支持使用 viper.SetDefault(key, value) 为 key 设置默认值 value，在没有通过配置文件、环境变量、 远程配置或命令行标志设置 key 所对应值的情况下

package main

import (
	"fmt"

	"github.com/spf13/viper"
)

func main() {
	// 设置默认配置
  
	viper.SetDefault("username", "wenl")
	viper.SetDefault("server", map[string]string{"ip": "127.0.0.1", "port": "8080"})

	// 读取配置值
	fmt.Printf("username: %s\n", viper.Get("Username")) // key 不区分大小写
	fmt.Printf("server: %+v\n", viper.Get("server"))
}

从配置文件读取配置

**Viper 可以搜索多个路径，但目前单个 Viper 实例只支持单个配置文件。Viper 不会默认配置任何搜索路径，将默认决定留给应用程序。

  • 当使用viper.SetConfigFile()函数来指定配置文件时，如果配置文件的文件名没有包含扩展名（例如.json、.yaml等），那么你需要使用viper.SetConfigType()函数来显式地指定配置文件的格式。这样viper才能正确地解析配置文件并加载相应的配置信息。

  • 通过 viper.AddConfigPath() 指定配置文件的搜索路径中，可以通过多次调用，来设置多个配置文件搜索路径。然后通过 viper.SetConfigName() 指定不带扩展名的配置文件，Viper 会根据所添加的路径顺序查找配置文件，如果找到就停止查找。

├─ main.go
|
|─config
    └──config //配置文件

package main

import (
	"fmt"

	"github.com/spf13/viper"
)

func main() {
	// 方式1 分开设置配置文件名和路径
	viper.SetConfigName("config") //指定配置文件
	viper.SetConfigType("yaml") //如果配置文件名中没有扩展名，则需要显式指定配置文件的格式  
    viper.AddConfigPath(".") //把当前目录加入到配置文件的搜索路径中  
	viper.AddConfigPath("./config") //可以添加多个
  // ====================================
  // 方法2 
  viper.SetConfigName("./config/config") //全路径
  viper.SetConfigType("yaml") //如果配置文件名中没有扩展名，则需要显式指定配置文件的格式 

	// 读取配置文件
	err := viper.ReadInConfig()
	if err != nil {
		fmt.Printf("无法读取配置文件: %s\n", err)
		return
	}
  // 成功读取配置文件...
}
