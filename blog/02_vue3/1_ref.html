什么是 ref

  • ref 和 reactive 一样,也是用来实现响应式数据的方法
  • ref 底层其实还是 reactive,所以当运行时系统会自动根据传入的 ref 转换成 reactive.
  • 在 vue 中使用 ref 的值不用通过 value 获取
  • 在 js 中使用 ref 的值必须通过 value 获取 👇
0 非响应式
0 响应式
ref
<template>
  <div>
    <div><xl-button @click="addA" type="primary">A++</xl-button>{{ A }} 非响应式</div>
    <!-- 在vue中使用ref的值不用通过value获取 -->
    <div><xl-button @click="addB" type="primary">B++</xl-button>{{ B }} 响应式</div>
    <div>
      <xl-button @click="showB" type="primary">查看B属性</xl-button>
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue'
// 定义A B
let A = 0
let B = ref(0)

function addA() {
  A++
  console.log(`当前A的值为：${A}`)
}
// 注意在js中使用ref的值必须通过value获取
function addB() {
  B.value++
  console.log(`当前B的值为：${B.value}`)
}
function showB() {
  console.log(B)
}
</script>
