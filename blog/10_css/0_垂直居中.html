利用定位

利用定位+margin:auto

父级设置为相对定位，子级绝对定位 ，并且四个定位属性的值都设置了0，那么这时候如果子级没有设置宽高，则会被拉开到和父级一样宽高
这里子元素设置了宽高，所以宽高会按照我们的设置来显示，但是实际上子级的虚拟占位已经撑满了整个父级，这时候再给它一个margin：auto它就可以上下左右都居中了

<template>
    <div class="father">
      <div class="son"></div>
    </div>
</template>
<style scoped>
  .father{
    position: relative;//定位
    width:500px;
    height:300px;
    border:1px solid #0a3b98;
  }
  .son{
    position: absolute;
    top:0;
    left:0;
    right:0;
    bottom:0;
    margin:auto;
    width:100px;
    height:40px;
    background: #f0a238;
  }
</style>

利用定位+margin:负值

这种方案不要求父元素的高度，也就是即使父元素的高度变化了，仍然可以保持在父元素的垂直居中位置，水平方向上是一样的操作
但是该方案需要知道子元素自身的宽高，但是我们可以通过下面transform属性进行移动

<template>
  <div class="father2">
    <div class="son2"></div>
  </div>
</template>

<style scoped>
 .father2 {
    position: relative;
    width: 300px;
    height: 300px;
    background: skyblue;
  }
  .son2 {
    position: absolute;
    top: 50%;
    left: 50%;
    margin-left:-50px; 
    margin-top:-50px;
    width: 100px;
    height: 100px;
    background: red;
  }
</style>

利用定位+transform

//translate(-50%, -50%)将会将元素位移自己宽度和高度的-50%
这种方法其实和最上面被否定掉的margin负值用法一样，可以说是margin负值的替代方案，并不需要知道自身元素的宽高

<template>
  <div class="father3">
    <div class="son3"></div>
  </div>
</template>

<style scoped>
.father3 {
  position: relative;
  width: 300px;
  height: 300px;
  background: #ddd;
}
.son3 {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%,-50%); 
  width: 100px;
  height: 100px;
  background: yellow;
}
</style>

布局

table布局

利用vertical和text-align可以让所有的行内块级元素水平垂直居中
设置父元素为display:table-cell，子元素设置 display: inline-block。

<template>
  <div class="father4">
    <div class="son4"></div>
  </div>
</template>

<style scoped>
.father4 {
   display: table-cell;
   width: 200px;
   height: 200px;
   background: #ffee00;
   vertical-align: middle;
   text-align: center;
}
.son4 {
  display: inline-block; //变成行内块级元素
  width: 100px;
  height: 100px;
  background: #999;
}
</style>

小结

上述方法中，不知道元素宽高大小仍能实现水平垂直居中的方法有：

  • 利用定位+margin:auto
  • 利用定位+transform
  • flex布局
  • grid布局

根据元素标签的性质，可以分为：

  • 内联元素居中布局
  • 块级元素居中布局

内联元素居中布局
水平居中

  • 行内元素可设置：text-align: center
  • flex布局设置父元素：display: flex; justify-content: center

垂直居中

  • 单行文本父元素确认高度：height === line-height
  • 多行文本父元素确认高度：display: table-cell; vertical-align: middle

块级元素居中布局
水平居中

  • 定宽: margin: 0 auto
  • 绝对定位+left:50%+margin:负自身一半

垂直居中

  • position: absolute设置left、top、margin-left、margin-top(定高)
  • display: table-cell
  • transform: translate(x, y)
  • flex(不定高，不定宽)
  • grid(不定高，不定宽)，兼容性相对比较差
